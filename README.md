# Pareva Jewels

A modern jewellery store website built with React, TypeScript, Vite, and Firebase.

## Features

- Responsive design for mobile, tablet, and desktop
- Jewellery product catalog
- Product categories
  - Gold Bangles
  - Bali
  - Bracelet
  - Chain
  - Ear Chain
  - Latkan
  - Mangalsutra Pendant
  - Sets
  - Nose Pins
  - Pendants
  - Rings
  - Tops
  - Jhumar
- Product search functionality
- Wishlist support
- Add to Cart functionality
- Admin product management
- Firebase integration
- Contact and store information

## Tech Stack

- React
- TypeScript
- Vite
- Firebase
- Firestore
- CSS/Tailwind CSS

## Installation

### Prerequisites

- Node.js (Latest LTS Version)
- npm

### Clone Repository

```bash
git clone https://github.com/Ayush-git-m/Pareva.git
cd Pareva
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env.local` file and add your Firebase/Gemini configuration:

```env
VITE_API_KEY=your_api_key
```

### Run Development Server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
